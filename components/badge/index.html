Badge

Badges are being used to display a notification count or status information.

Image Avatar with Status Badge

This library has 3 types of status badges that can be integrated with Avatars. You can show colors to show the online status of user. You can show different badge status by adding class online, offline, dnd. You can also set different badge placement as you desired by adding combination of classes top, bottom, left, right.

Text Initial Avatar with Status Badge

Similar to image avatar, you can also show badge status for text initial avatar.

Icon with Number Badge

This library also provides number badge on icon. You can get number badge on all types of icon like button, link or simple icon. Number badge comes in different sizes. You can get different sizes by adding classes small, medium, large.
