Avatar

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Avatar with Image

Avatar is available in 3 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size small, medium, large (e.g. class="avatar small").

Avatar with Text Initials

You can use initial letters of user in Avatar as well. Instead of providing image just write text initial.
