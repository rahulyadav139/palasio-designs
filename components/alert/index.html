Alert

Alerts are used to attract user's attention for important information without interrupting the user's flow.

Standard Alert

There are 4 different types of alert available in this library. Add specific class to get desire alert. Classes are info, success, update and danger. You can customize alert what you want with setting different icon with different alert. Put icon under alert-icon class span. You can copy html part from below code snippet.

                 //alert dismiss functionality

const btnDismiss = document.querySelector('.btn-dismiss-alert');

  alert.style.display = 'none';
