## Quick Start

To start using the components in your project, Copy-paste the stylesheet `<link>` into your `<head>` before all other stylesheets to load our CSS.

```html
    <link
        rel="stylesheet"
        href="https://palasio-designs.netlify.app/style.css"
    />
```
